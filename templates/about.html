ActiDispenses

Une interface de travail pour piloter les dispenses et diagnostiquer Oracle.

L'application centralise la consultation des dispenses, le monitoring des flux, les outils de diagnostic SQL et la gestion des credentials Oracle dans une interface unique.

Ce que fait l'application
  • Recherche de dispenses et de personnes
  • Suivi mensuel des statuts et anomalies
  • Interrogation SQL Oracle en lecture seule
  • Gestion de plusieurs credentials Oracle

Modules

Dispenses

Requetes SQL

Comptes Oracle

Principes

Lecture seule la ou c'est sensible

Secrets hors du code
Les cles applicatives et Oracle sont chargees depuis l'environnement.
Auditabilite
Les executions SQL sensibles sont historisees et exportables.
Robustesse locale
Tests, checks, commandes de diagnostic et monitoring permettent un usage plus fiable.
